# GitHub Presentation Guide

## Recommended repository name

`canadian-banking-sql-analytics`

## Recommended project title

**Canadian Big 5 Banking SQL Analytics**

## Recommended GitHub description

SQL portfolio project using PostgreSQL and synthetic Canadian banking data to analyze Big 5 bank performance, customer risk, fraud alerts, branch targets, campaigns, loans, cards, service SLAs, and data quality with advanced SQL, views, materialized views, procedures, and Docker.
